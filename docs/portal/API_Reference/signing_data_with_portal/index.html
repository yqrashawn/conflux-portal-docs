Signing Data with ConfluxPortal

#History of the signing methods

Portal is forked from MetaMask and there are multiple signing methods in MetaMask. You can read the history of all the signing methods in MetaMask documentation.

#What changed in ConfluxPortal

  • We changed cfx_sign to use the personal_sign logic. We recommend using cfx_sign method instead of personal_sign.
  • The signTypedData, signTypedData_v1, signTypedData_v3, signTypedData_v4 still works the same way as MetaMask. But we are planning to change them before our mainet launch.
